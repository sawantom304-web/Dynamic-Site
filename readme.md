# Motion — Dynamic Site

A small, responsive landing-page prototype built with plain HTML and CSS.

## Summary

Motion is a minimal static site that demonstrates a simple hero layout, mockup graphics, and responsive styles.

## Files

- `Motion.html` — main HTML page (entry point)
- `style.css` — site styles and responsive rules

## Preview

Open `Motion.html` in a browser, or serve the folder locally:

```bash
# from the project root
python3 -m http.server 8000
# then open http://localhost:8000/Motion.html
```

## Notes for maintainers

- Built with plain HTML/CSS (no build step).
- Fonts are loaded from Google Fonts (Inter).
